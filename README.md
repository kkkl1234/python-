
本项目聚焦摩洛哥丹吉尔市电力消耗精准预测，为区域能源调度与资源优化提供支撑。以 2017 年 3 个配电网区域能耗数据及 5 项环境特征（温度、湿度等）为基础，针对传统 ARIMA 模型泛化能力不足的痛点，构建 Transformer-CNN 混合模型，探索单指标、多特征、多区域三类建模方式。最终多区域联合建模方案表现最优，通过跨区域协同学习捕捉关联规律，实现平均 MAPE 2.22% 的高精度预测，有效适配能耗非平稳波动与突变特征，显著优于传统线性模型。
